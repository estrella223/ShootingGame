# 🎮ShootingGame

### 📝정책<br/>
1. 스테이지 존재 - 스테이지마다 적의 이미지 변경,
                  스테이지가 높아질수록 적의 수가 많아짐
2. 이벤트 - 두번째, 세번째 스테이지에 제거할 수 없는 적 삽입
3. 적(향수, 담배, 술)이 랜덤한 위치에서 랜덤한 방향으로 떨어짐
4. 스테이지별로 최고점수 제한, 점수가 일정수준 도달시 다음 스테이지로 이동
5. bgm과 효과음 삽입
